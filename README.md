# 🌐 Dildon J — Developer Portfolio

A personal portfolio website built with pure HTML, CSS, and JavaScript.
Live at 👉 **[dildon-portfolio.netlify.app](https://dildon-portfolio.netlify.app)**

---

## ✨ Features

- Glassmorphism design with animated background
- Fully responsive (mobile + desktop)
- Smooth scroll reveal animations
- Animated skill progress bars
- Working contact form
- Projects showcase with GitHub links

---

## 🛠️ Built With

- HTML5
- CSS3 (Glassmorphism, Flexbox, Grid, Animations)
- JavaScript (DOM, IntersectionObserver, Events)
- Google Fonts (Space Grotesk + Inter)
- Deployed on Netlify

---

## 📁 Folder Structure

```
portfolio/
├── index.html
├── style.css
└── script.js
```

---

## 🚀 Run Locally

Just open `index.html` in any browser — no setup needed!
